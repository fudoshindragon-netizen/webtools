# MicroVenture Labs — Web Tools

A collection of lightweight, single-purpose web utilities for content creators.

## Tools

### 1. 📝 Markdown to HTML Converter
Convert Markdown to HTML instantly. Live preview, copy, and export.

- **Live preview** as you type
- **Copy to clipboard** (full HTML document)
- **Download** as standalone .html file
- **Dark theme** with full Markdown rendering
- **Pro version** ($9): PDF export, batch convert, custom CSS themes, API access

### 2. 🎬 YouTube Title & Hashtag Optimizer
Generate optimized video titles and trending hashtags for any topic.

- **5 title suggestions** from 20+ proven template patterns
- **10 trending hashtags** with view volumes across 10 niches
- **Copy All** buttons for titles and hashtags
- **Click to copy** individual items
- **Pro version** ($12): 50 titles, competitor analysis, thumbnail tips, CSV export

## Tech Stack

- Vite + vanilla JavaScript
- Zero backend dependencies
- Client-side only — $0 hosting cost
- Dark theme across all tools

## Getting Started

```bash
# Pick a tool
cd md-to-html
# or
cd yt-title-optimizer

# Install dependencies
npm install

# Development
npm run dev

# Production build
npm run build
```
